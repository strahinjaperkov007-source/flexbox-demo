Stavka 3 je prikazana poslednja zbog order: 2.

Ključni CSS kod:

.flex-container {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
  justify-content: space-around;
  align-items: center;
  gap: 15px;
}

.item {
  flex: 1 1 80px;
  align-self: auto;
}

.item3 {
  order: 2;
  align-self: flex-end;
}

.item4 {
  flex-grow: 2;
}

Objašnjenje glavnih razlika

  • Raspored: Flexbox automatski raspoređuje stavke; redosled se može menjati sa order.
  • Vertikalno centriranje: Flexbox koristi align-items i align-self; standardni CSS koristi vertical-align.
  • Prelazak u novi red: Flexbox sa flex-wrap automatski prelama stavke; standardni CSS ne.
  • Rast/skupljanje: Flexbox stavke mogu proporcionalno da rastu/skupljaju; standardni CSS ne.
  • Razmak između stavki: Flexbox koristi gap; standardni CSS margine.
